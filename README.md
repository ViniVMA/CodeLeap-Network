## Inicie o projeto localmente

Clone the repo

```bash
  git clone https://github.com/ViniVMA/CodeLeap-Network
```

Go to the repo directory

```bash
  cd CodeLeap-Network
```

Install the dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
